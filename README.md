💼 Student Library Management System
Tech Stack: Java, Spring Boot, REST API, MySQL, Hibernate, Postman, Maven

Description:
Developed a backend system to manage student and library interactions, including book issuing, returns, student records, and inventory management. The system exposes RESTful APIs for all core functionalities and integrates with a relational database for persistent storage.

Key Responsibilities:

Designed and implemented RESTful APIs using Spring Boot for CRUD operations on students, books, and transactions.

Integrated Hibernate ORM for database operations and ensured data consistency with MySQL.

Developed layered architecture (Controller-Service-Repository) for better maintainability and scalability.

Implemented exception handling, validation, and logging across the application.

Used Postman to test APIs and verify response accuracy and performance.

Followed best practices for code structuring, documentation, and dependency management using Maven.
